# PolitiSimple 🇫🇷

**L'information politique claire, accessible et transparente.**

PolitiSimple est une plateforme web moderne conçue pour simplifier la compréhension des programmes politiques et des parcours judiciaires des personnalités publiques françaises pour l'horizon 2026.

## ✨ Fonctionnalités

- **Base de données évolutive** : Profils de personnalités nationales et maires de grandes métropoles.
- **Programmes structurés** : 6 catégories clés par candidat (Économie, Sécurité, Social, Droits des Femmes, LGBT+, Environnement).
- **Transparence judiciaire** : Accès direct aux dossiers et statuts juridiques avec sources citées.
- **Recherche Intelligente** : Filtrage en temps réel par nom, parti, ville ou contenu biographique.
- **Navigation Optimisée** : Routage par hash pour une expérience fluide avec l'historique du navigateur.

## 🚀 Installation & Développement

1.  **Cloner le dépôt** :
    ```bash
    git clone https://github.com/votre-user/politisimple.git
    cd politisimple
    ```

2.  **Installer les dépendances** :
    ```bash
    npm install
    ```

3.  **Lancer en mode développement** :
    ```bash
    npm run dev
    ```

4.  **Compiler pour la production** :
    ```bash
    npm run build
    ```

## 🛠️ Stack Technique

- **Frontend** : React 18
- **Build Tool** : Vite
- **Styling** : Tailwind CSS 4
- **Icons** : Lucide React
- **Data** : JSON statique pour une rapidité optimale
